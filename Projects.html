Nethys Web Scrapper

Show More

Written in JavaScript and using Node.JS, I developed this webstite scrapper to scrape creature data from the website Archives of Nethys, an online repository for the tabletop RPG game Pathfinder. This data is then outputed into a .CMX file used by the software Combat Manager, which was developed to make running combat in Pathfinder easier.

Originally, Combat Manager was developed for the 1st edition of Pathfinder. When the new edition arrived, Combat Manager did not release an edition in response. So instead of losing the feature-rich functionality of Combat Manager, I wrote this script to make importing 2nd edition monsters much easier.
